# Сессии

Сюда записывай итоги рабочих сессий: что сделано, что решено, на чём остановился.

## Формат записи сессии

Создавай файл с названием вида `2026-04-02 краткое описание.md`:

```markdown
---
tags: [session]
date: 2026-04-02
---

# 2026-04-02 — Название сессии

## Сделано
- ...

## Решения принятые сегодня
- ...

## Осталось на следующую сессию
- ...

## Ссылки
- [[заметка к которой обращался]]
```

## Архив

_Сессии появятся здесь после первой рабочей встречи._
