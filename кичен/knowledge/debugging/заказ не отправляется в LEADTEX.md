# Заказ не отправляется в LEADTEX

## Диагностика

Открой DevTools → Network при оформлении заказа. В консоли должно появиться:

```
📤 Отправка заказа в LEADTEX
🆔 Telegram ID (из localStorage): 123456789
🆔 Используемый Telegram ID: 123456789
📦 Payload для отправки: {...}
📡 Отправка запроса на URL: /api/webhook
```

После ответа сервера:
- `✅ Заказ успешно отправлен в LEADTEX` — запрос прошёл
- `❌ Ошибка отправки в LEADTEX: 4xx/5xx ...` — ошибка на стороне LEADTEX

## Возможные причины и решения

### 1. Контакт не существует в LEADTEX

**Симптом**: статус 200, но в LEADTEX ничего не появилось.

**Причина**: LEADTEX ищет контакт по `telegram_id`. Если пользователь ещё не писал боту — контакта нет.

**Решение**: Попроси пользователя написать `/start` боту. Только после этого LEADTEX создаёт контакт и может принять заказ.

### 2. telegram_id не определён

**Симптом**: в консоли `Telegram ID (из localStorage): null` и используется `123456789` (mock ID).

**Решение**: Убедись, что приложение открыто через Telegram, а не в обычном браузере. Если тестируешь — включи `DEV_MODE: true` в `js/config.js`. Подробнее: [[telegram_id не определяется в приложении]]

### 3. CORS ошибка (ERR_FAILED на /api/webhook)

**Симптом**: в консоли ошибка fetch, статус сети = failed.

**Причина**: Скорее всего, приложение запущено локально (не через Vercel), и serverless-функция недоступна.

**Решение для локальной разработки**: Запусти через `npm run dev` и используй DEV_MODE, либо задеплой на Vercel и тестируй там.

### 4. Неверный UUID вебхука LEADTEX

**Симптом**: статус 404 или 401 от LEADTEX.

**Решение**: Проверь URL в двух местах:
- `api/webhook.js:11` — строка `const WEBHOOK_URL = '...'`
- `vercel.json` → `rewrites[0].destination`

Оба должны содержать актуальный UUID из настроек LEADTEX.

### 5. Форма не проходит валидацию

**Симптом**: `submitOrder()` не доходит до `sendOrderToServer()`.

**Проверь**: имя, телефон (11 цифр после маски), город, адрес — все поля обязательны. Телефон должен содержать 11 цифр: `phoneInput.value.replace(/\D/g, '').length < 11`.

## Быстрая проверка из консоли

```javascript
// Проверить, что telegram_id есть
localStorage.getItem('telegram_id');

// Проверить, что корзина не пуста
console.log(cart);

// Симулировать отправку заказа
sendOrderToServer({
  id: 'TEST-001',
  items: [{ id: 'k1', name: 'Тест', price: 100, quantity: 1, total: 100 }],
  total: 100,
  customer: { name: 'Тест', phone: '+7 (999) 000-00-00', email: '' },
  delivery: { city: 'Москва', address: 'ул. Теста, 1' },
  comment: '',
  timestamp: new Date().toISOString()
});
```
