# Конфигурация-ориентированное проектирование

## Суть паттерна

Весь контент магазина — товары, категории, настройки — хранится в единственном объекте `CONFIG` в `js/config.js`. Чтобы изменить ассортимент, не нужно трогать логику приложения.

## Структура CONFIG

```javascript
const CONFIG = {
    WEBHOOK_URL: '/api/webhook',   // куда отправлять заказы

    SHOP: {
        name: 'Keychain Shop',
        logo: '🔑',
        currency: '₽',
        currencyCode: 'RUB',
        phone: '+7 (999) 123-45-67'
    },

    CATALOG: {
        itemsPerPage: 12,
        maxCartItems: 50,
        allowPreOrder: true
    },

    CATEGORIES: [
        { id: 'all', name: 'Все товары', icon: '✨' },
        { id: 'classic', name: 'Классические', icon: '🗝️' },
        // ...
    ],

    PRODUCTS: [
        {
            id: 'keychain-classic-metal',
            name: 'Классический металлический',
            price: 290,
            category: 'classic',
            description: '...',
            image: 'https://...',
            stock: 100,
            rating: 4.8
        },
        // ещё 14 товаров...
    ],

    DEV_MODE: false,
    MOCK_USER: { id: 123456789, first_name: 'Тест', ... }
};
```

## Вспомогательные методы CONFIG

| Метод | Использование |
|---|---|
| `CONFIG.getProductById(id)` | `showProduct()` в app.js |
| `CONFIG.getProductsByCategory(catId)` | `renderProducts()` при фильтрации |
| `CONFIG.formatPrice(price)` | Везде где выводится цена |
| `CONFIG.formatRating(rating)` | Карточки товаров и детали |

## Как добавить товар

Добавить объект в `PRODUCTS`:
```javascript
{
    id: 'keychain-new-001',          // уникальный ID
    name: 'Новый брелок',
    price: 990,
    category: 'classic',             // должна совпадать с id в CATEGORIES
    description: 'Описание товара',
    image: 'https://images.unsplash.com/...?w=400&q=80',
    stock: 50,
    rating: 4.5
}
```

Пересборка не нужна — сайт статический.

## Двойной экспорт

Файл работает и в браузере (глобальная переменная `CONFIG`), и в тестах Node.js:

```javascript
if (typeof module !== 'undefined' && module.exports) {
    module.exports = CONFIG;
}
```
