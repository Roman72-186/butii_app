- Максимум: **9 990 ₽** (Бриллиантовый)

- Наименьший склад: премиум (5–15 шт.)

## Поля товара в config.js

```javascript
{
    id: string,           // уникальный, формат: keychain-{категория}-{тип}
    name: string,
    price: number,        // в рублях, целое число
    category: string,     // совпадает с id в CATEGORIES
    description: string,
    image: string,        // URL Unsplash с параметрами ?w=400&q=80
    stock: number,
    rating: number        // 4.5 – 5.0
}
```
