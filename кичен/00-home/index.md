## Структура кодовой базы

```
keychain-shop/
├── index.html          # Единственная HTML-страница (SPA)
├── js/
│   ├── config.js       # Меню, категории, цены, доставка, точки самовывоза
│   ├── max-app.js      # Обёртка над MAX Web App SDK
│   └── app.js          # Витрина, корзина, оформление, оплата, статусы
├── css/style.css       # CSS переменные, темы, адаптив
├── server.js           # Express API магазина
├── server/             # Auth, catalog, orders, payments, CRM
├── vercel.json         # Маршрутизация и заголовки безопасности
├── tests/              # Jest-тесты (jsdom)
└── bridge/             # Legacy bridge для атрибуции, не часть витрины
```

---

## Команды разработки

```bash
npm run dev          # Локальный сервер на :3000
npm test             # Запустить тесты Jest
npm run test:watch   # Тесты в режиме наблюдения
npm run test:coverage # Отчёт покрытия в coverage/
vercel --prod        # Деплой в продакшн
```
